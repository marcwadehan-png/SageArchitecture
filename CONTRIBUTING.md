# CONTRIBUTING.md

## 贡献指南

感谢您对 SageArchitecture 项目的关注！

## 如何贡献

1. **Fork 本仓库**
2. **创建特性分支** (`git checkout -b feature/AmazingFeature`)
3. **提交更改** (`git commit -m 'Add some AmazingFeature'`)
4. **推送到分支** (`git push origin feature/AmazingFeature`)
5. **创建 Pull Request**

## 开发规范

- 代码风格：遵循 PEP 8
- 提交信息：使用清晰的中文描述
- 测试：新增功能需包含测试用例

## 问题反馈

请通过 GitHub Issues 提交问题报告。
